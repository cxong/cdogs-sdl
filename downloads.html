Linux packages for cdogs-sdl

C-Dogs SDL is available through various Linux distribution package managers. Installing through your distribution's package manager is recommended over the generic Linux binary.

Fedora 42

Package: cdogs-sdl-0.7.3-14.fc42.rpm

Install with:

sudo dnf install cdogs-sdl
Ubuntu 25.04 LTS

Package: cdogs-sdl_2.0.0-1_amd64.deb

Install with:

sudo apt install cdogs-sdl
Arch Linux (AUR)

Package: cdogs

Install with an AUR helper (e.g., yay):

yay -S cdogs

Or manually from AUR:

git clone https://aur.archlinux.org/cdogs.git
cd cdogs
makepkg -si
